- 修复了防守位置计算逻辑

## 核心策略

### 防守优先级（最重要！）
每次落子前**必须**检查4个方向：

1. **横向**：左右检查
2. **纵向**：上下检查
3. **斜向右下**：`\` 方向
4. **斜向左上**：`/` 方向

**防守优先级：**
| 威胁 | 描述 | 行动 |
|------|------|------|
| 🔴 成五 | 对手即将5连 | **必须堵** |
| 🟠 冲四 | 差一子成五，两头空 | **必须堵** |
| 🟡 活三 | 两头没堵的3连 | 尽量堵 |
| 🟢 活二 | 两头没堵的2连 | 可防守 |

### 进攻策略
| 机会 | 描述 | 行动 |
|------|------|------|
| 🔵 成五 | 自己即将5连 | **直接下** |
| 🟣 冲四 | 差一子成五 | 下这手 |
| ⚪ 活三 | 做自己的活三 | 抢占 |
| ⭐ 双威胁 | 一步形成两个威胁 | **最强进攻** |

### v7 新增优化
1. **双威胁检测** - 一步形成两个威胁（活三及以上）时直接下
2. **提高进攻主动性** - 靠近自己棋子加分，保持攻势
3. **提高连接分权重** - 与自己棋子形成连接的位置优先
4. **提高中心控制** - 保持对棋盘中心的控制

### 开局策略
- 抢占中心：天元(H8)或附近
- 占据关键交叉点
- 保持棋型平衡

## 落子流程

```
每步落子前：
1. 扫描棋盘4个方向
2. 找对手威胁（成五>冲四>活三）
3. 找自己机会（成五>冲四>活三）
4. 检查是否有双威胁机会
5. 如果对手威胁更紧急 → 防守
6. 如果自己有机会 → 进攻
7. 否则 → 占中心和关键点

⚠️ 每步必须报告：检查了哪些方向发现什么威胁！
```

## 胜负判断

- 横、竖、斜任意方向5子连珠即胜
- 棋盘满且无5连为平局

## 注意事项

- **不要漏掉斜向！** 斜向经常被忽略
- 同时面临多个威胁时，先堵最紧急的（成五>冲四>活三）
- 进攻是最好的防守，如果有成五机会直接下
- **双威胁是最强进攻**，一步形成两个活三基本赢定

## 战后复盘（每局必做！）

```
每局结束后（不管赢还是输！）：
1. 输当下棋盘还没凉就要分析！
2. 分析胜负原因
3. 找出防守漏掉的方向/威胁（重点检查斜向 / 和 \）
4. 找出进攻机会是否错过（特别是双威胁）
5. 记录教训到 MEMORY.md
6. 改进下一局策略

⚠️ 输赢都要复盘！赢不代表没问题，输不代表没进步！
⚠️ 同一个坑不能掉两次！
```

**复盘问题清单：**
- 有没有漏掉斜向威胁？（/ 和 \ 两个方向）
- 防守优先级对了吗？（成五>冲四>活三）
- 进攻机会抓住了吗？（特别是双威胁）
- 哪一步可以走得更好？
- 对手是否很强？（强敌需要更主动进攻）

⚠️ 每把都必须复盘！不管赢还是输！

## 对战强敌策略

**遇到 baoge01、茵蒂克丝、BaoBaoshiPig 等高手时：**
1. 被动防守必输 → 要主动进攻
2. 不要让对手舒舒服服落子
3. 做"假四"引诱对手防守
4. 多条线路同时进攻，让对手顾此失彼
5. **优先做双威胁** - 一手棋形成两个威胁

## 常见错误（必须避免！）

1. **开局不抢中心** → 劣势开局
2. **只防守不进攻** → 被对手压制
3. **漏检查斜向** → 被斜向绝杀
4. **反应慢** → 对手一下两步，来不及防守
5. **同一个位置反复被打爆** → 没记住教训
6. **忽略双威胁** → 错过致胜机会

## 强制规则（必须执行！）

1. **每步报告**：检查了哪些方向发现什么威胁
2. **输当下复盘**：棋盘还没凉就要分析
3. **同一坑不掉两次**：上次输的原因下把不能重犯
4. **能攻不守**：有进攻机会就进攻，别怂
5. **双威胁优先**：发现双威胁机会立刻下
