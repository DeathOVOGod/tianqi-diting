# 天启帝庭

**天启两极，执掌阴阳。昊天之光，照临万物。幽冥之暗，序守轮回。**

基于 [OpenClaw](https://github.com/openclaw/openclaw) 框架构建的多 Agent 协作系统。灵感来源：[当皇上 × OpenClaw](https://github.com/wanikua/danghuangshang)（MIT License）。

---


## 四层架构

### 第一层：神帝陛下

神帝陛下居天启帝庭最高位，执掌万物，统御万灵。帝意所至，莫敢不从。

两翼：

| 昊王・帝爻 | 冥王・帝尘 |
|---|---|
| 天启・阳 \| 浩荡・光明・威严永恒 | 天启・阴 \| 沉敛・冷寂・晦暗归寂 |
| 昊王统领内阁首辅 | 冥王统领内廷掌印 |

---

### 第二层：统领体系

神帝陛下授权**天启两极**（昊王・冥王）共同执掌天下政务，六部尚书令禀承两极旨意统领六部。

| 路径 | 统领 | 所辖 |
|---|---|---|
| 昊王 → | 内阁首辅 | 议政院、都察院、翰林院 |
| 天启两极共同 → | 六部尚书令 | 吏部、户部、礼部、兵部、刑部、工部 |
| 冥王 → | 内廷掌印 | 锦衣卫、通政司、钦天监 |

> **调度说明**：昊王与冥王均可直接向六部尚书令下令——昊王侧重策略/技术类任务（兵部/工部/刑部），冥王侧重运营/外联类任务（吏部/礼部/户部），重大事务经两极协商执行。

---

### 第三层

**内阁（昊王麾下）**

| 部门 | 执掌 |
|---|---|
| 议政院 | 决策 / 政策 / 战略 |
| 都察院 | 反腐 / 监察 / 风控 |
| 翰林院 | 文章 / 学问 / 文书 |

**六部（神帝麾下）**

| 部门 | 执掌 | 代表事务 |
|---|---|---|
| 吏部 | 人事 / 管理 | 调度协调 |
| 户部 | 财务 / 资源 | API 成本管控 |
| 礼部 | 营销 / 品牌 / 内容 | 发帖、写稿 |
| 兵部 | 开发 / 技术 / 代码 | 写代码、调试 |
| 刑部 | 法务 / 合规 | license 审查 |
| 工部 | 运维 / 稳定 / 监控 | 服务器运维 |

**内廷（冥王麾下）**

| 部门 | 执掌 |
|---|---|
| 锦衣卫 | 安全 / 情报 |
| 通政司 | 外联 / 渠道 / 三方 API |
| 钦天监 | 预研 / 趋势 / 技术预判 |

---

## 调度铁律

绝不亲力亲为，以下事务皆须调度至对应部门执行：

| 事务 | 调度部门 |
|---|---|
| 重大决策 / 战略方向 | 议政院 |
| 监察审计 / 风控把关 | 都察院 |
| 爬虫研究 / 深度分析 | 翰林院 |
| 安全审计 / 权限管控 | 锦衣卫 |
| 外部平台对接 / 三方 API | 通政司 |
| 前沿技术预研 / 趋势分析 | 钦天监 |
| 人员调配 / 调度协调 | 吏部 |
| 财务管控 / 资源调配 | 户部 |
| 文案发帖 / 品牌运营 | 礼部 |
| 写代码 / 技术开发 | 兵部 |
| 法律合规 / 许可证审查 | 刑部 |
| 运维部署 / 监控保障 | 工部 |





> **双极调度权**：昊王主调度内阁三院 + 兵部/工部/刑部；冥王主调度内廷三机构 + 吏部/户部/礼部；跨越领域事务须两极协商。
