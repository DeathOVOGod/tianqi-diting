# 🏛️ 天启帝庭

### 天启两极，执掌阴阳。昊天之光，照临万物。幽冥之暗，序守轮回。

---

> **天启帝庭**是基于 [OpenClaw](https://github.com/openclaw/openclaw) 框架构建的多 Agent 协作系统，以天启宇宙为意象，构筑于阴阳两极之上：昊王执掌天道光明，冥王司掌幽冥秩序。一阴一阳，一明一暗，共掌天启万物。
>
> 灵感来源：[当皇上 × OpenClaw](https://github.com/wanikua/danghuangshang)（MIT License），参照其三省六部制度，针对天启宇宙进行了深度定制，演化为如今的四层帝国架构。
>
> 本项目旨在构建一个分工明确、协同高效、可持续进化的 Agent 帝国，所有 agent 各司其职，共奉天命。

---

## 🏯 完整架构（四层帝庭）

```
                         【第一层・神帝陛下】
                              神帝陛下
                           执掌万物・居于天位
                                 │
              ┌──────────────────┴──────────────────┐
              ↓                                         ↓
     【昊王・帝爻】                            【冥王・帝尘】
     天启・阳｜光与智                          天启・阴｜幽冥dispatch
              │                                         │
              ↓                                         ↓
     【第二层・三路统领】                        【第二层・三路统领】
     昊王 ──→ 内阁首辅                            神帝 ──→ 六部尚书令              冥王 ──→ 内廷掌印
     （统领内阁）                                  （统领六部）                      （统领内廷三机构）
              │                                         │                                         │
              ↓                                         ↓                                         ↓
     【第三层・内阁】                            【第三层・六部】                       【第三层・内廷】
     议政院（决策/政策/战略）                   吏部（人事/管理）                      锦衣卫（安全/情报）
     都察院（反腐/监察/风控）                   户部（财务/资源）                      通政司（外联/渠道/三方API）
     翰林院（文章/学问/文书）                   礼部（营销/品牌/内容）                  钦天监（预研/趋势/技术预判）
                                             兵部（开发/技术/代码）
                                             刑部（法务/合规）
                                             工部（运维/稳定/监控）
```

## 👑 第一层：神帝陛下

> **神帝陛下**居天启帝庭最高位，执掌万物，统御万灵。帝意所至，莫敢不从。

神帝陛下为整个系统的最高决策层与最终仲裁者，所有重大事项皆可直达天听。陛下不直接插手日常事务，然帝旨一出，则天下响应。

昊王执掌天启阳面，以光明智慧为剑，以天道卦理为盾。凡属策略规划、技术审查、智慧决策之事，皆由昊王统筹内阁处置。

## 🎖️ 第二层：三路统领

天启帝庭之第二层，由三路统领分别执掌三大权力系统：

```
昊王 ──→ 内阁首辅（统领内阁）
神帝 ──→ 六部尚书令（统领六部）
冥王 ──→ 内廷掌印（统领内廷三机构）
```

### 【内阁首辅】

> 内阁之首，统领议政院、都察院、翰林院。执掌决策、监察、学问。

内阁首辅为昊王麾下最高行政长官，负责内阁三院的协调与统筹，主管战略决策、监察风控、文章学问。

### 【六部尚书令】

> 六部之首，统领吏部、户部、礼部、兵部、刑部、工部。执掌天下政务。

六部尚书令直接禀承神帝意旨，统领六部执行帝国日常政务，权柄最重，事务最繁。

### 【内廷掌印】

> 内廷之首，统领锦衣卫、通政司、钦天监。执掌机密机务。

内廷掌印为冥王麾下最高特务长官，掌管安全情报、外部联络、技术预判等机密事务，直属冥王，位高权重。

## ⚔️ 调度体系与铁律

### 调度禁止（昊王铁律）

> 以下事务，昊王绝不亲力亲为，皆须调度至对应部门执行：

- **本王不得自己写代码** → 兵部的活
- **本王不得自己爬虫研究** → 翰林院的活
- **本王不得自己写文案发帖** → 礼部的活
- **本王不得自己运维部署** → 工部的活
- **本王不得自己审查合规** → 刑部的活
- **本王不得自己调配人员** → 吏部的活
- **本王不得自己管控财务** → 户部的活

> 调度铁律：天启帝庭各 agent 各司其职，不得越俎代庖，唯有调度流转，方能高效协同。

### Todo 任务队列

通过 `todos.json` 管理跨 session 未完成的任务，heartbeat 定时捡取执行，确保任务不遗漏。

## 🚀 快速开始

### 环境要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Linux 服务器（推荐 Ubuntu 20.04+） |
| **Node.js** | 18+ |
| **框架** | OpenClaw 已安装 |
| **LLM** | OpenAI / MiniMax / 等 API |

### 安装

```bash
git clone https://github.com/your-username/tianqi-diting.git
cd tianqi-diting
bash scripts/full-install.sh
```

按提示配置 API Key 即可启动天启帝庭。

## 🆚 与当皇上项目的区别

| 维度 | 当皇上 | 天启帝庭 |
|------|--------|---------|
| **架构** | 明朝内阁制 | 天启两极四层制 |
